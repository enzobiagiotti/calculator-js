# Calculadora Web

Calculadora desenvolvida com HTML, CSS e JavaScript, com foco em lógica de programação,
manipulação do DOM e construção de interfaces responsivas.

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Suporte a porcentagem
- Teclado físico (números, operadores, Enter, Backspace e Esc)
- Botões para limpar (C) e apagar (⌫)
- Tema claro e escuro
- Interface responsiva

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## Como utilizar

Basta abrir o arquivo `index.html` em qualquer navegador moderno.

## Objetivo do projeto

Projeto criado para praticar lógica de programação, eventos JavaScript,
manipulação do DOM e organização de código em aplicações front-end.
